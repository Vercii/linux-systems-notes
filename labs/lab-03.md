# Linux File System Commands
In this lab exercise, I learned how to use File System Commands.

---

Practicing file system navigation. Moving between the root directory (/), the system binary directory (/usr/bin), and using shortcuts like cd and cd ~ to return home, while testing home directory expansions using echo ~username.
![Screenshot](/assets/lab-03-1.png)

Exploring paths and hidden items. Navigating up and down the directory tree using relative path shortcuts (.. and ../dict), listing standard user folders, and using ls -a to reveal hidden configuration files (like .bashrc).
![Screenshot](/assets/lab-03-2.png)

Using ls flags and pattern matching in /etc. Running a long listing of hosts, recursively listing contents with ls -R, using * to find files starting with "s", ???? to filter for exactly 4-character names, and [abcd]* to match specific starting letters.
