# Globbing
In this lab exercise, I took the time to get comfortable with Globbing and manipulating files through CLI commands.

---
Demonstrating Linux shell globbing and wildcard patterns (*, ?, and []) with the echo command to filter and match file names in the current directory.

![Screenshot](/assets/lab-04-1.png)

Practicing basic file copying (cp) and file deletion (rm), including the use of the verbose flag (cp -v) and specifying the current directory (.) as a destination.

![Screenshot](/assets/lab-04-2.png)

Preserving file attributes (timestamps/permissions) during copies using cp -p, renaming a file during a copy, and recursively copying a directory structure with cp -R.

![Screenshot](/assets/lab-04-3.png)

Verifying a recursive directory structure using ls -lR, handling a syntax error where ls was mistakenly typed before rm, and successfully removing a directory using rm -r.

![Screenshot](/assets/lab-04-4.png)

Creating an empty file with touch, renaming/moving it using the mv command, and deleting it using rm.
